# Babysitting Schedule Validator

This repository contains my solution for Take Home Exam 2 of the course **CENG111 – Introduction to Computer Engineering Concepts** at **Middle East Technical University (METU)**.

## Description

The program evaluates whether a proposed monthly babysitting schedule satisfies a set of predefined constraints.

Each caregiver has specific availability and restrictions. The program verifies whether the schedule violates any of these rules. If the schedule is valid, it calculates the total babysitting cost for the month.

## Topics Covered

- Python programming
- Constraint validation
- List processing
- Functional programming techniques

## Files

### solution.py
Contains the implementation that:
- validates schedule constraints
- checks caregiver availability
- computes the total babysitting cost when the schedule is valid
