# Take Home Exam 3

This folder contains my solution for **Take Home Exam 3** of the course
**CENG111 – Introduction to Computer Engineering Concepts** at **Middle East Technical University (METU)**.

## Description

This assignment focuses on computing the overlapping area between two geometric shapes in 2D space: a convex quadrilateral and a triangle.

The program calculates the intersection area of these shapes using their vertex coordinates.

## Topics Covered

* Python programming
* Basic computational geometry
* Coordinate-based calculations
* Geometric reasoning
