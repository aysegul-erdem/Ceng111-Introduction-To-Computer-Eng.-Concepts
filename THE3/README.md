# Convex Quadrilateral–Triangle Intersection Area

This repository contains my solution for **Take Home Exam 3** of the course  
**CENG111 – Introduction to Computer Engineering Concepts** at  
**Middle East Technical University (METU)**.

## Description

This program computes the intersection area between two geometric shapes in 2D space: a **convex quadrilateral** and a **triangle**.

Given the vertex coordinates of both shapes, the program determines their overlapping region and calculates its area using coordinate-based geometric calculations.

## Topics Covered

- Python programming
- Computational geometry
- Polygon intersection
- Coordinate-based geometric calculations

## Files

solution.py  
Contains the implementation that computes the intersection area of the given shapes.
