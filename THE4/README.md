# Take Home Exam 4

This folder contains my solution for **Take Home Exam 4** of the course
**CENG111 – Introduction to Computer Engineering Concepts** at **Middle East Technical University (METU)**.

## Description

This assignment focuses on working with a partial family tree representation and deriving different kinship relations.

The program processes a nested list representation of a family tree and determines relationships such as siblings, aunts, uncles, and cousins for a given person.

## Topics Covered

* Python programming
* Tree-like data structures
* Recursive and iterative reasoning
* Relationship queries in hierarchical data
