# Babysitting Schedule Validator

This folder contains my solution for **Take Home Exam 2** of the course
**CENG111 – Introduction to Computer Engineering Concepts** at **Middle East Technical University (METU)**.

## Description

This assignment focuses on implementing a program that evaluates whether a proposed monthly babysitting schedule satisfies a set of predefined constraints.

The program checks if the schedule violates any rules related to the availability and restrictions of different caregivers. If the schedule is valid, it calculates the total babysitting cost for the month.

## Topics Covered

* Python programming
* Rule validation and constraint checking
* List processing
* Functional programming techniques

## Files

```text
solution.py
```

This file contains the implementation of the function that validates the schedule and computes the total cost when the schedule is valid.
